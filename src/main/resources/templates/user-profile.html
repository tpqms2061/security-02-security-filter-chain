🔗 이 페이지 접근 시 실행된 필터들

  1. SecurityContextHolderFilter - 인증 정보 로드
  2. UsernamePasswordAuthenticationFilter - 로그인 처리
  3. AnonymousAuthenticationFilter - 익명 사용자 체크 (Skip)
  4. AuthorizationFilter - USER 권한 확인 ✅
  5. 커스텀 로깅 필터들 - 요청/응답 로깅

💡 콘솔에서 "🔍 [FILTER]" 로그를 확인하여 실제 필터 실행 순서를 관찰해보세요!
