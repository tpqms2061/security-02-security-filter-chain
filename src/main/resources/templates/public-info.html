🌍 공개 정보 페이지

인증 없이 접근 가능한 페이지

🔓 공개 접근 권한

이 페이지는 로그인 없이도 접근 가능합니다. Spring Security에서 /public/** 경로로 설정되어 있어 인증이 필요하지 않습니다.

📋 이 페이지에서 실행되는 필터들

  • SecurityContextHolderFilter - 보안 컨텍스트 관리
  • AnonymousAuthenticationFilter - 익명 사용자로 처리
  • AuthorizationFilter - 권한 검사 (통과 ✅)
  • 커스텀 로깅 필터들 - 요청 로깅

🚀 빠른 접근

인증 과정 없이 바로 접근 가능하여 로딩 속도가 빠릅니다.

📊 공개 정보

회사 소개, 제품 정보, 연락처 등 공개적으로 제공되는 정보를 확인할 수 있습니다.

🔍 SEO 친화적

검색 엔진이 쉽게 접근할 수 있어 SEO 최적화에 유리합니다.

🛡️ 보안 필터 체인 동작 확인

브라우저 개발자 도구의 Network 탭에서 이 페이지 요청 시 Status 200 OK가 바로 반환되는 것을 확인할 수 있습니다. 로그인 페이지로 리다이렉트되지 않습니다.
